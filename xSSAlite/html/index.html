xSSAlite

xSSAlite is a Mathematica package that implements Dan Gillespie's Stochastic Simulation Algorithm using Cellerator-arrow notation. The "lite" part of xSSAlite refers to the fact that this version only implements the Cellerator-mass action reactions. Non-mass-action reactions are only implemented in the full version which is currently still in testing.

The original Stochastic Simulation Algorithm was published in the paper "Exact Stochastic Simulation of Coupled Chemical Reactions," The Journal of Physical Chemistry, 81(25):2340-2361 (1977).
