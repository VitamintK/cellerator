The SSALite Palette xSSA Home

Description

The SSAlite Palette can be used to generate templates for reactions in a Mathematica notebook

If the palette is installed in the Mathematica/SystemFiles/FrontEnd/Palettes directory then it can be opened from the Palettes menu.

Scrolling over any item in the palette gives a short synopsis:

Clicking on any item provides a template in the current notebook that can be filled in.
