DeleteCell Cellzilla2D Home

Description

Removes a cell from a DTissue object. No vertices or edges are removed from the tissue.

Needs["Cellzilla2D`"];

Return Value

RemoveCell[DTissue, i] - returns a DTissue with cell i removed.

To find out which vertices and edges are actually used by the DTissue see EdgesUsed and VerticesUsed. To find out the numbers of the cells actually used in a DTissue T use

First/@TissueCells[T]
