UndangleVertices Cellzilla2D Home

Description

Removes all the dangling vertices from a tissue. A vertex is considered dangling if it is not part of any edge.

Needs["Cellzilla2D`"];

Return Value

UndangleVertices[tissue] - returns the modified tissue.
