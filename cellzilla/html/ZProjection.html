ZProjection
ShowZProjection		 Cellzilla2D Home

Description

Drops the z-coordinates of a 3-d tissue. Assumes the vertex coordinates are all given in 3-Dimensions. This is useful in projecting raw 3D data onto a 2D template.

Needs["Cellzilla2D`"];

Return Value

ZProjection[T] returns the modified tissue.

ShowZProjection[T] converts the tissue to 2D and returns a plot. Same options as ShowTissue.

T is the original tissue
