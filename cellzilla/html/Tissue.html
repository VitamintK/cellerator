Description

Tissue is an unistantiated tissue that r represents a tissue composed of vertices, edges, and cells.

Needs["Cellzilla2D`"];

Return Value

Tissue[v, e, c] represents the tissue with vertices at coordinates v; edge list e; and cell list c.

The edge list e is list of integer vertex number pairs e={{i1,j1}, {i2, j2}, ...}. If {i,j} is in e then it means that there is an edge between vertex i and vertex j, where i and j refer to the ith and jth coordinate pairs in the list v.

The cell list c is a list of cells, where each cell is a list of edges. Thus if c = {c1, c2, c3, ...} where c3={4, 7, 9 ,12, 32} that means that cell 3 is composed of edges 4, 7, 9, 12, 32 in the edge list e.
