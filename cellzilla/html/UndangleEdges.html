UndangleEdges Cellzilla2D Home

Description

Removes all the dangling edges from a tissue. An edge is considered dangling if it is not part of any cell.

Needs["Cellzilla2D`"];

Return Value

UndangleEdges[tissue] - returns the modified tissue.
