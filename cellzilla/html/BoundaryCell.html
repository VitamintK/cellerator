BoundaryCell Cellzilla2D Home

Description

Returns a data structure that can be used to build a single cell consisting of the entire tissue.

Needs["Cellzilla2D`"];

Return Value

BoundaryCell[Tissue] returns a data list
{"Coordinates"->{{x1,y1},{x2,y2},....}, (* coordinates of bounding vertices *)
"Edges"->{e1,e2,...}, (* edge numbers of bounding edges *)
"Vertices"->{v1,v2,...} } (* vertex numbers of vertices on boundary *)
