BoundaryCell Cellzilla2D Home

Description

Returns a data structure that can be used to build a single cell consisting of the entire tissue.

Needs["Cellzilla2D`"];

Return Value

BoundaryCell[Tissue] returns a data list {"Coordinates"->{{x1,y1},{x2,y2},....}, "Edges"->{e1,e2,...}, "Vertices"->{v1,v2,...} } where e1,e2,... are edge numbers, and v1,v2,... are vertex numbers in the original tissue, and {{x1,y1},...} are the coordinates of the vertices on the bounding cell.

Options

Example

To create a tissue consisting of a single cell that is equivalent to the boundary of the original cell:


bc="Edges"/.BoundaryCell[T]
v=TissueVertices[T]
e=TissueEdges[T]
Tnew=Tissue[v,e,bc]
