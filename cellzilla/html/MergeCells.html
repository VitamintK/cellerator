MergeCells Cellzilla2D Home

Description

Needs["Cellzilla2D`"];

Merges cells together.

Return Value

MergeCells[tissue, {i, j, k, ...}] returns a new tissue object with the specified cells merged into a single cell.

NOTE: The cells must be neighbors; this is NOT CHECKED. If the cells are not neighbors, unexpected results will occur. In particular, while all of the original cells will be deleted, the shape of the newly merged cells will contain only a partial subset of the desired cells.
