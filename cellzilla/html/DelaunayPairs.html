DelaunayPairs Cellzilla2D Home

Description

Determines the Delaunay Triangulation and returns it as a list of number pairs, where each number is an integer representing a vertex.

Needs["Cellzilla2D`"];

Return Value

DelaunayPairs[{{x1,y1}, {x2, y2}, ...}] returns a list of integer pairs {{i1,j1}, {i2, j2}, ...} where each {i,j} means that vertex i is connected to vertex j in the Delaunay Triangulation.
