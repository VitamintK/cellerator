Description

Needs["Cellzilla2D`"];
Needs["xSSA`"];

Covert a Cellerator or xSSA reaction network to a network of xSSA reactions on a multicellualr template.

Return Value

SSANetwork[network,diff,tissue,verbose] - returns an SSA network on the tissue from a single cell xSSA network.

xSSANetwork[network,diff,tissue,verbose] - returns an SSA network on the tissue from a single cell xlr8r network. The input reaction network is preprocessed by xSSA`SSAtoXLR8R

In both functions, an xSSA network is returned. The only difference is that the first fucntion takes as input xSSA reactions, and the second takes xlr8r reactions.
