perimeterLength Cellzilla2D Home

Note: Previously called Perimeter in Mathematican versions 7-9. The name was changed because of a conflict with a newly defined built-in of the same name in Mathematica version 10.

Description

Needs["Cellzilla2D`"];

Return Value

perimeterLength[{v1,v2,...}]
returns the perimeter of polygon with the specified vertices.

perimeterLength[tissue,i]
returns the perimeter of cell i in the specified Tissue object

perimeterLength[tissue]
returns a list of the perimeters of all the cells in the specified Tissue object
