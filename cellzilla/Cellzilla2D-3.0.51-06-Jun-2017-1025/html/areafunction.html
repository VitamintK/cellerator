areafunction Cellzilla2D Home

Note: previously called Area in Mathematican versions 7-9. The name was changed because of a conflict with a newly defined built-in of the same name in Mathematica version 10.

Description

Needs["Cellzilla2D`"];

Return Value

areafunction[{v1,v2,...}]
returns the area of polygon with the specified vertices.

areafunction[tissue,i]
returns the area of cell i in the specified Tissue object

areafunction[tissue]
returns a list of the areas of all the cells in the specified Tissue object
