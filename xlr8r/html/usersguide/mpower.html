mPower Reference Home

mPower is a Mathematica wrapper for Qhull, regtet, and pwrvtx. These are open source C and Fortran programs that compute Vornoi and Power Diagrams in various dimensions.

The mPower package is not used directly by any currently existing software in the xCellerator ensemble. However, during development, some of these functions were used but have since been removed. The package remains for users who would like to continue to have access.
