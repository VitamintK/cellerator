xlr8r:CelleratorML

LatexForm CelleratorML Home

Translates a CelleratorML file to LaTeX.

Needs["CelleratorML`"];
Needs["xlr8r`"];

Return Value

LatexForm[inputfile, outputfile]

returns the name of the file generate

If the specified file already exists, a new (unique) file name is generated.
